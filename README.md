# EDA Analytics Projects

Exploratory Data Analysis (EDA) projects focused on uncovering insights from real-world datasets.

## Projects

| Project | Dataset | Notebook |
|---|---|---|
| [Netflix](netflix/) | Netflix Movies & TV Shows | [Netflix.ipynb](netflix/notebooks/Netflix.ipynb) |
| [Students Performance](students-performance/) | Students Performance in Exams | [Students_performance.ipynb](students-performance/notebooks/Students_performance.ipynb) |
| [Titanic](titanic/) | Titanic passenger manifest | [titanic.ipynb](titanic/notebooks/titanic.ipynb) |
